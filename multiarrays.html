Multi-Dimenstional Arrays

n this tutorial, you learned about arrays. These arrays are called one-dimensional arrays.
In the next tutorial, you will learn about multidimensional arrays (array of an array).
Syntax : data-type array_name [size_rows][size_column];
Declration : float x[3][4];
Initialization of a 2d array
int c[2][3] = {{1, 3, 0}, {-1, 5, 9}};
